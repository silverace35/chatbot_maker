# CYNA Products and Services Catalog

## Customer Segments

- **SMB** (PME): Small and medium businesses
- **Mid-Market** (ETI): Mid-sized companies
- **MSP**: Managed Service Providers

---

## Available Services

### PREVENTION

#### Standard Security Audit (S-AUDIT-STD)
- **Category**: PREVENTION
- **Type**: One-time intervention
- **Description**: Comprehensive security audit to assess IT system posture. Covers policies, processes, and configurations.
- **Deliverables**: Detailed audit report (PDF), Risk mapping, Prioritized action plan
- **For**: SMB, Mid-Market
- **Can be sold standalone**: Yes

#### Advanced 360° Security Audit (S-AUDIT-AV)
- **Category**: PREVENTION
- **Type**: One-time intervention
- **Description**: 360° audit including organizational review, log analysis, and business interviews (CISO, CIO).
- **Deliverables**: 360° audit report, Management summary, Detailed action plan
- **For**: SMB, Mid-Market
- **Can be sold standalone**: Yes

#### Web Application Pentest (S-PENT-WEB)
- **Category**: PREVENTION
- **Type**: One-time intervention
- **Description**: Attack simulation (black/grey box) on exposed web application, with vulnerability report and recommendations.
- **Deliverables**: Vulnerability report, Exploitation POC, Remediation recommendations
- **For**: SMB, Mid-Market
- **Can be sold standalone**: Yes

#### Infrastructure Pentest (S-PENT-INFRA)
- **Category**: PREVENTION
- **Type**: One-time intervention
- **Description**: Infrastructure robustness testing (internal/external): Active Directory, VPN, firewalls, internet exposure.
- **Deliverables**: Infrastructure vulnerability report, Risk summary, Detailed recommendations
- **For**: SMB, Mid-Market
- **Can be sold standalone**: Yes

### PROTECTION

#### 24/7 Managed SOC for SMB/Mid-Market (S-SOC-PME)
- **Category**: PROTECTION
- **Type**: Recurring (subscription)
- **Description**: 24/7 supervision of work environment (workstations, servers, mobile, users) via EDR and Digital Workplace, with detection, analysis, and incident response.
- **Deliverables**: Alert handling, Monthly reports, Dashboard via platform
- **For**: SMB, Mid-Market
- **SLA**: 24/7
- **Can be sold standalone**: No (included in SOC offers)

#### 24/7 SOC for MSP (S-SOC-MSP)
- **Category**: PLATFORM PROTECTION
- **Type**: Recurring (subscription)
- **Description**: 24/7 SOC for MSP with multi-tenant model, EDR/Digital Workplace integration, and dedicated MSP console.
- **Deliverables**: Multi-client alert handling, Consolidated MSP reports, MSP console access
- **For**: MSP
- **SLA**: 24/7
- **Can be sold standalone**: No

### INCIDENT RESPONSE

#### 24/7 Incident Response - CERT (S-CERT-IR)
- **Category**: RESPONSE
- **Type**: Recurring
- **Description**: Investigation, containment, eradication, remediation, and post-mortem for security incidents, with 24/7 availability.
- **Deliverables**: Incident report, Remediation plan, Post-mortem summary
- **For**: SMB, Mid-Market, MSP
- **SLA**: 24/7
- **Can be sold standalone**: Yes
