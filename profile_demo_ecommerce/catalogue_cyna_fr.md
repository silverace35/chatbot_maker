# Catalogue des produits et services CYNA

## Segments de clientèle

- **PME** : Petites et moyennes entreprises
- **ETI** : Entreprises de taille intermédiaire  
- **MSP** : Managed Service Providers / Infogérants

---

## Services disponibles

### PRÉVENTION

#### Audit de sécurité standard (S-AUDIT-STD)
- **Catégorie** : PRÉVENTION
- **Type** : Intervention ponctuelle (ONE_SHOT)
- **Description** : Audit global de sécurité pour évaluer la posture du SI. Couvre politiques, processus et configurations.
- **Livrables** : Rapport d'audit détaillé (PDF), Cartographie des risques, Plan d'actions priorisé
- **Pour** : PME, ETI
- **Peut être vendu seul** : Oui

#### Audit de sécurité avancé 360° (S-AUDIT-AV)
- **Catégorie** : PRÉVENTION
- **Type** : Intervention ponctuelle (ONE_SHOT)
- **Description** : Audit 360° incluant revue organisationnelle, analyse de logs et interviews métiers (RSSI, DSI).
- **Livrables** : Rapport d'audit 360°, Synthèse management, Plan d'actions détaillé
- **Pour** : PME, ETI
- **Peut être vendu seul** : Oui

#### Pentest applicatif Web (S-PENT-WEB)
- **Catégorie** : PRÉVENTION
- **Type** : Intervention ponctuelle (ONE_SHOT)
- **Description** : Simulation d'attaques (black/grey box) sur une application web exposée, avec rapport de vulnérabilités et recommandations.
- **Livrables** : Rapport de vulnérabilités, POC d'exploitation, Recommandations de remédiation
- **Pour** : PME, ETI
- **Peut être vendu seul** : Oui

#### Pentest infrastructure (S-PENT-INFRA)
- **Catégorie** : PRÉVENTION
- **Type** : Intervention ponctuelle (ONE_SHOT)
- **Description** : Test de robustesse de l'infrastructure interne et/ou externe : Active Directory, VPN, pare-feux, exposition internet.
- **Livrables** : Rapport de vulnérabilités infra, Synthèse risques, Recommandations détaillées
- **Pour** : PME, ETI
- **Peut être vendu seul** : Oui

### PROTECTION

#### SOC managé 24/7 PME/ETI (S-SOC-PME)
- **Catégorie** : PROTECTION
- **Type** : Récurrent (abonnement)
- **Description** : Supervision 24/7 de l'environnement de travail (postes, serveurs, mobiles, utilisateurs) via EDR et Digital Workplace, avec détection, analyse et réponse aux incidents.
- **Livrables** : Traitement des alertes, Rapports mensuels, Tableau de bord via plateforme
- **Pour** : PME, ETI
- **SLA** : 24/7
- **Peut être vendu seul** : Non (inclus dans les offres SOC)

#### SOC 24/7 pour MSP (S-SOC-MSP)
- **Catégorie** : PROTECTION PLATEFORME
- **Type** : Récurrent (abonnement)
- **Description** : SOC 24/7 pour MSP avec modèle multi-tenant, intégration EDR/Digital Workplace et console MSP dédiée.
- **Livrables** : Traitement des alertes multi-clients, Rapports consolidés MSP, Accès console MSP
- **Pour** : MSP
- **SLA** : 24/7
- **Peut être vendu seul** : Non

### RÉPONSE À INCIDENT

#### Réponse à incident 24/7 - CERT (S-CERT-IR)
- **Catégorie** : RÉPONSE
- **Type** : Récurrent
- **Description** : Investigation, confinement, éradication, remédiation et post-mortem en cas d'incident de sécurité, avec disponibilité 24/7.
- **Livrables** : Rapport d'incident, Plan de remédiation, Compte-rendu post-mortem
- **Pour** : PME, ETI, MSP
- **SLA** : 24/7
- **Peut être vendu seul** : Oui

#### Préparation & conseil CERT (S-CERT-PREP)
- **Catégorie** : PRÉVENTION ACCOMPAGNEMENT
- **Type** : Récurrent
- **Description** : Revues de crise, mise à jour de runbooks, briefings menaces et rapports trimestriels CERT.
- **Livrables** : Runbooks à jour, Rapports trimestriels CERT, Synthèses menaces
- **Pour** : PME, ETI, MSP
- **Peut être vendu seul** : Oui

### ACCOMPAGNEMENT

#### Accompagnement subventions cyber (S-SUBV-ASSIST)
- **Catégorie** : ACCOMPAGNEMENT
- **Type** : Intervention ponctuelle
- **Description** : Accompagnement pour mobiliser les financements publics/para-publics (Diag Cyber, Chèque Cyber, etc.).
- **Livrables** : Dossier de subvention préparé, Conseils sur les dispositifs disponibles
- **Pour** : PME, ETI
- **Peut être vendu seul** : Oui

#### Sensibilisation & formation utilisateurs (S-TRAIN-AWARE)
- **Catégorie** : ACCOMPAGNEMENT
- **Type** : Récurrent
- **Description** : Sessions de formation (présentiel/visio), contenus de sensibilisation et simulateur de phishing.
- **Livrables** : Supports de formation, Rapports de campagne de phishing, Attestations de participation
- **Pour** : PME, ETI, MSP
- **Peut être vendu seul** : Oui
